Howto for documentation
--------------------------------

Create an XML file containing the documentation page, 

put it in the docentries.xml file as a tocitem element
	- url attribute refers to the xml file
	- target attribute refers to the java documentation element name
	
	the text informations are build from the page title ...
	
launch the build target for constructing the product help ....
refresh the docbuild element in the project
to view the help result : launch the view help target in the build.xml ant script
