Internationalized web site

This site is maintained in two language French and English, i whished not having to manage it with a dynamic language (php, ...) and wish to have the two languages in the same page to maintain it more easily. In this case, a simple solution has been used : XML. With a XSL stylesheet i can filter the language. The xml offer also to ease the manipulation of the content by bots (robots) for automatic translation by google or babelfish.

Web site organisation

The organisation is simple, two subfolder are used for the localized pages (/en fo english, /fr for french). Common items are put in the /common subfolder.

Content definition in an XML file

Each page is written in an xml file (using editix) this xml is in XHTML format.
